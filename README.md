# Aplicação de Ensino de Programação e Tecnologias para Pessoas Surdas

Este projeto é uma aplicação desenvolvida para o ensino de programação e tecnologias, com foco na inclusão de pessoas surdas. A aplicação utiliza trechos de códigos explicado em Libras (Língua Brasileira de Sinais) para proporcionar uma experiência de aprendizado acessível e eficaz.
